## Supported Features
* Validator Keys Creation and Management - `make run-cli validator keys --help`
* Validator Registration and Status check - `make run-cli validator --help`
* Reward Collecting - `make run-cli rewards --help`

## Running `layer-cli` from source

```bash
git clone https://github.com/ton-link/layer-cli.git
cd layer-cli
make install
make run-cli
```
